# Dependencies

- `bash`, `curl`, `tar`

# Install

Plugin:

```shell
asdf plugin add please https://github.com/asdf-community/asdf-please.git
```

Check [asdf](https://github.com/asdf-vm/asdf) readme for more instructions on how to
install & manage versions.
